## 🛠️ What It Does

Logo Viking is a free online image toolbox built for designers, creators, marketers and developers. Every tool runs **100% in your browser** — your files never leave your device.

### Tools Available

| Category | Tools |
|----------|-------|
| **Optimize** | Compress Image, AI Upscale, Remove Background |
| **Edit** | Resize, Crop, Rotate, Watermark |
| **Convert** | To JPG, To PNG, To WebP, To AVIF |
| **Create** | Meme Generator, Color Palette Extractor, Screenshot Beautifier |
| **Security** | Blur Faces, Remove EXIF Data |
| **Export** | Image to PDF |

## 🚀 How to Use

### Option 1 — Visit the live site
Go to **[logoviking.com](https://logoviking.com)** and use any tool instantly.

### Option 2 — Run locally
```bash
git clone https://github.com/yourusername/logoviking.git
cd logoviking
python3 -m http.server 8080
```
Then open `http://localhost:8080`
