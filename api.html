Logo Viking API

The Logo Viking API lets you integrate all 16 image processing tools into your applications. Built on REST with JSON responses. Available on Pro and Business plans.

🔑 Base URL: https://api.logoviking.com/v1

Authentication

Authenticate by passing your API key in the Authorization header of every request.

# Include in all requests
Authorization: Bearer lv_live_your_api_key_here

Find your API key in your account dashboard under Settings → API.

Compress Image

POST /compress
Compress an image while maintaining visual quality.
ParameterTypeRequiredDescription
fileFileRequiredImage file (max 200MB on Pro)
qualityIntegerOptionalQuality 1-100 (default: 80)
output_formatStringOptionaljpeg, png, webp (default: original)
 
const formData = new FormData();
formData.append('file', imageFile);
formData.append('quality', 80);
formData.append('output_format', 'webp');

const response = await fetch('https://api.logoviking.com/v1/compress', {
  method: 'POST',
  headers: { Authorization: 'Bearer YOUR_KEY' },
  body: formData
});

const { url, original_size, compressed_size, savings_pct } = await response.json();

Resize Image

POST /resize
Resize an image to specified dimensions.
// Example response
{
  "url": "https://cdn.logoviking.com/results/abc123.jpg",
  "original_size": 204800,
  "new_size": 56320,
  "width": 1920,
  "height": 1080,
  "expires_at": "2026-03-26T02:00:00Z"
}

Node.js SDK

npm install logoviking-api
const LogoViking = require('logoviking-api');
const pf = new LogoViking({ apiKey: 'lv_live_...' });

// Compress
const result = await pf.compress('./photo.jpg', { quality: 80 });

// Resize
const resized = await pf.resize('./photo.jpg', { width: 1200, height: 800 });

// Remove Background
const nobg = await pf.removeBg('./photo.jpg');

Python SDK

pip install logoviking
from logoviking import LogoViking

result = pf.compress("photo.jpg", quality=80)
